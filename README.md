# Rana Gallery 🎨

A dedicated art portfolio and gallery website designed to showcase creative works. This project features a public-facing gallery for visitors and an administrative dashboard to manage content.

🔗 **Live Demo:** [artist-rana-gallery.vercel.app](https://artist-rana-gallery.vercel.app)

## 📖 About

**Rana Gallery** is a web-based platform built to display art collections in an elegant, responsive layout. It includes tools for managing the gallery, allowing for dynamic updates to the artwork displayed without needing to modify the code directly for every new piece.

## ✨ Features

* **Responsive Gallery:** A clean, grid-based layout to showcase artwork (`index.html`).
* **Admin Dashboard:** A central hub to manage gallery content (`dashboard.html`).
* **Dynamic Content Management:** Interface to add new art pieces/cards easily (`addCard.html`).
* **Modern Styling:** Custom CSS for a unique and artistic aesthetic.

## 🛠️ Technologies Used

* **HTML5** - Structure and layout.
* **CSS3** - Styling, grid layouts, and responsiveness.
* **JavaScript** - DOM manipulation and logic for adding cards and gallery interactivity.
